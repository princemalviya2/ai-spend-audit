# AI Spend Audit Dashboard

## Overview
AI Spend Audit Dashboard helps users analyze and optimize spending on AI tools.

## Features
- Upload statement
- AI tool selection
- Dashboard cards
- Spending chart
- AI recommendations
- Result summary

## Tech Stack
- Next.js
- React
- Tailwind CSS
- Recharts

## Installation

```bash
npm install
npm run dev
```
